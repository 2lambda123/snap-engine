VISAT 2.3 ROI extensions

Overview

  • The "File/Export" menu now contains a new command "Export ROI Pixels" which can be used to export the values of all pixels contained in a ROI either to the clipboard or a text file.
  • The "Analysis" menu now contains a new command "Compute ROI Area" which can be used to compute the ROI area in square kilometers.
  • The ROI Definition Window now contains a new command "Assign ROI to other products" which is used to transfer the ROI to other open products. ROI shapes are automatically transformed into the satellite coordinate spaces of the target products.
