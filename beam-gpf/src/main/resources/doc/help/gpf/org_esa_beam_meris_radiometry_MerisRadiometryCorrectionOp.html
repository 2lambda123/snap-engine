Meris.CorrectRadiometry Operator

Meris.CorrectRadiometry Operator Description

Overview

Name:Meris.CorrectRadiometry
Full name:org.esa.beam.meris.radiometry.MerisRadiometryCorrectionOp
Purpose:Performs radiometric corrections on MERIS L1b data products.
Version:1.0

Description

No description available.

Sources

Name Description
sourceProduct The source product.

Parameters

Name Type Description Default
doCalibration boolean Whether to perform the calibration. true
sourceRacFile File The radiometric correction auxiliary file for the source product.
targetRacFile File The radiometric correction auxiliary file for the target product.
doSmile boolean Whether to perform Smile-effect correction. true
doEqualization boolean Perform removal of detector-to-detector systematic radiometric differences in MERIS L1b data products. true
reproVersion ReprocessingVersion The version of the reprocessing the product comes from. Is only used if equalisation is enabled. AUTO_DETECT
doRadToRefl boolean Whether to perform radiance-to-reflectance conversion. false
