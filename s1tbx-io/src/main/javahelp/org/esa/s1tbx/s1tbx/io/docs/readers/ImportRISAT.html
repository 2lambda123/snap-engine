Import RISAT-1

Import RISAT-1 Product

This option allows to import RISAT-1 products into SNAP.

The supported CEOS RISAT-1 product types are:

  • SLC Single Look Complex
  • GRD Ground Range Detected
  • Terrain Georeferenced
  • Enhance Terrain Georeferenced
The supported modes are:
  • HRS High Resolution Spotlight
  • FRS Fine Resolution Stripmap
  • MRS Medium Resolution ScanSAR
  • CRS Coarse Resolution ScanSAR

in single, dual, circular or quad polarizations
