New in BEAM 4.7
  • VISAT now has a "real" Geometry Management allowing to draw and manipulate geometries and import them from ESRI Shapefiles.
  • The VISAT Mask and ROI Management has been completely revised. Any number of raster data masks can be defined by band arithmetic expressions, value ranges and from existing geometry. Each mask can serve as ROI for raster data analysis.
