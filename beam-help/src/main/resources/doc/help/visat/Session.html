Session

A session can stores your current configuration of VISAT. If you open a saved session all products, scene views and layers will be restored. VISAT will be in state as if you have never closed the session.
A session file has the file extension .beam.

From the file menu you can
save, open and close a session.

If you store the your opened products beside the session file, the file pathes to the products are stored as relative pathes. This means that you can take the parent folder of the session file and move somewhere else on your hard disk without breaking the references top the products. This allows you to package the whole folder and send it e.g. by e-mail, too. If the recipient of this e-mail opens the session, he will have the same VISAT configuration has you had as you stored the session.
