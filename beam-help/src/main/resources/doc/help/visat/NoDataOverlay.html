No-Data Overlay

   No-Data Overlay

Toggles the overlay of the no-data mask within a band's image view. The overlay style properties can be modified using the no-nata overlay page in the preferences dialog.

The no-data mask overlays displays all the band's pixels with undefined sample values. In order to control no-data pixels, use the band's no-data value and/or the band's valid pixel expression in the property editor for selected band.
