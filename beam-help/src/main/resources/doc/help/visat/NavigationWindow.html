The Navigation Window

The Navigation Window is used to move the viewport of an image view and to zoom in and out of it. The current viewport is depicted by a semi-transparent rectangle which can be dragged in order to meve the viewport to another location. It also provides a slider used to zoom in and out of the view:

The Navigation window additionally provides the following features via its tool buttons:

Zoom In: Zooms in by a factor of 1.2.

Zoom Out: Zooms out by a factor of 1/1.2.

Zoom Actual Pixel: Sets the zoom factor to the default value so that the size of an image pixel has the same size of a display pixel.

Zoom All: Adjusts the viewport to cover the entire image.

Synchronise: Synchronises the viewports of all compatible image views.
