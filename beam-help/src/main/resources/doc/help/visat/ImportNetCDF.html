Import NetCDF

The NetCDF reader enables VISAT to import NetCDF data products.

The NetCDF reader supports any image-like NetCDF file structure up to netCDf version 3.5. The currently used NetCDf Java library version is 2.2.12.

Note: You can find additional information about NetCDF at http://www.unidata.ucar.edu/software/netcdf/.
