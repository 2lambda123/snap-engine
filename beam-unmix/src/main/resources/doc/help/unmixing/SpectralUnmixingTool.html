Spectral Unmixing Tool

Introduction

The spectral unmixing tool is used to decompose a source spectrum into a set of given endmembers. The result of the unmixing is a measure of the membership of each endmember to the source source spectrum. This measure is called the endmember's abundance. The tool can also compute an error measure for each spectral source band and a summary, root mean square error band.

The abundances can be computed using three different algorithms:

  1. Unconstrained LSU - The abundances are unconstrained and can become any numeric value.
  2. Constrained LSU - The sum of abundances equals 1. This is the default unmixing algorithm.
  3. Fully Constrained LSU - The sum of abundances equals 1 and abundance values cannot be less than zero.
